# 🚀 Task Manager App (Full Stack)

A modern full-stack Task Manager web application with authentication, task management, and a clean UI.

---

## 🌐 Live Demo

🔗 Frontend: https://your-app.vercel.app  
🔗 Backend: https://your-backend.onrender.com  

---

## ✨ Features

- 🔐 User Authentication (JWT)
- 📝 Create Tasks
- ✏️ Edit Tasks
- ❌ Delete Tasks
- ✅ Mark Tasks as Completed
- 📊 Task Statistics (Total & Completed)
- 🔒 Protected Routes
- 🎨 Responsive UI with Tailwind CSS

---

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Bcrypt (Password Hashing)
